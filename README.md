# Synent-Task1-Datacleaning-Olatunji-Ayomide
# Titanic Data Cleaning & Preprocessing

# Problem Statement
Clean and prepare the Titanic dataset for analysis by handling missing values, removing duplicates, converting data types, and renaming columns.

# Dataset
Titanic Dataset - sourced from datasciencedojo on GitHub

# Approach
- Filled missing Age and Fare values with median
- Dropped Cabin column (too many missing values)
- Checked and confirmed no duplicate rows
- Converted Survived, Pclass, Sex, Embarked to category type
- Renamed all columns to clean, readable format

# Results
Clean dataset with 418 rows and 11 columns, ready for analysis
